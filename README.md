## About

This project discovers an artist's top tracks and related artists on Spotify.

## Technologies

Project is created with:

- React v16.13.1
- React Animate On Scroll v2.1.5
- Animate.css v4.1.0

## Setup

To run this project, install it locally using npm:

```
$ cd artist-discoverer
$ npm install
$ npm start
```
